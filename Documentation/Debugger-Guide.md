Step 1: Enter Play Mode

Click Play button in Unity
Wait for all systems to initialize (check Console for [ServiceContainer] Container ready message)

Step 2: Add Items Using Context Menu

From Project Window

Navigate to your item assets in Project window (e.g., Assets/Data/Items/Potions/)
Select any item asset (Potion, Material, Weapon, Gear)
Right-click the selected asset
You'll see a menu: Debug Item â–¶

Choose an option:

Add 5x to Bag - Adds 5 items to player's bag (main inventory)
Add 10x to Storage - Adds 10 items to storage room
Use Potion - Uses the potion on current player (Potions only)
Equip Weapon - Equips weapon on player (Weapons only)
Roll Weapon Stats - Randomizes bonus stats (Weapons/Gear only)
